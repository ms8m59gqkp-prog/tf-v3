## 검증 방법

외부 전문가는 이 문서 세트를 다음과 같이 사용합니다:

1. **V3 기능 구현 후** → 해당 도메인 spec 파일을 열고
2. **각 항목(상태 전이, 워크플로우 단계, 타입 정의 등)이 V3에 존재하는지** 1:1 대조
3. **누락/변경된 부분**을 기록하여 V3 팀에 피드백

모든 내용은 V2 코드에서 직접 추출한 사실(fact)이며, 추론(inference)은 포함되지 않습니다.
